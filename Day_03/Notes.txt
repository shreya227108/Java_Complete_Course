1.Operators in Java
-Operators are special symbols in Java that perform operations on variables and values.
They are like the "action symbols" of the language.

    1. Arithmetic Operators
    Used for mathematical operations: + , - , * , / , %

    2. Relational (Comparison) Operators
    Used to compare values (result = true/false): == , != , > , < , >= , <=

    3. Logical Operators
    Used to combine conditions: && (AND), || (OR), ! (NOT)

    4. Assignment Operators
    Used to assign values to variables: = , += , -= , *= , /= , %=

    5. Unary Operators
    Work with a single operand: + , - , ++ , -- , !

    6. Bitwise Operators
    Work on bits (0s and 1s): & , | , ^ , ~ , << , >> , >>>

    7. Ternary Operator (?:)
    Shortcut for if-else: condition ? value_if_true : value_if_false

    8. instanceof Operator
    Checks whether an object is an instance of a class.

2.Expressions in Java
-An expression is a combination of variables, constants, operators, and method calls that evaluates to a single value.

------------------------------------------------------------------------------------------------------------------------

3.Conditional Statements in Java

1.if Statement
-Executes a block of code only if the condition is true.

2.if-else Statement
-Executes one block if condition is true, otherwise another block.

3.if-else-if Ladder
-Used when there are multiple conditions.

4.Nested if
-An if inside another if.

5.switch Statement
-Used when you want to compare a value against multiple fixed options (instead of multiple if-else).

#Quick Summary Table
| Statement         | When to Use                              |
| ------------------| ---------------------------------------- |
| if                | Single condition                         |
| if-else           | Two possible outcomes                    |
| if-else-if ladder | Multiple conditions                      |
| nested if         | One condition depends on another         |
| switch            | Multiple choices based on a single value |
